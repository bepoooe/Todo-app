## üéØ How to Use

### Adding Tasks
1. Enter your task description in the main input field
2. Select priority level (Low, Medium, High)
3. Choose a category (Personal, Work, Shopping, Health, Other)
4. Optionally set a due date
5. Click "Add Task" or press Ctrl/Cmd + Enter

### Managing Tasks
- **Complete**: Click the circle checkbox to mark tasks as complete
- **Delete**: Click the trash icon to remove tasks
- **Filter**: Use the filter buttons to view specific task types
- **Search**: Type in the search box to find specific tasks

### Filtering Options
- **All Tasks**: Show everything
- **Pending**: Show only incomplete tasks
- **Completed**: Show only finished tasks
- **High Priority**: Show only high-priority tasks
